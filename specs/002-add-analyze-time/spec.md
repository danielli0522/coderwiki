# Feature: 代码质量分析列表页面添加分析日期

**Feature Branch**: `002-add-analyze-time`
**Created**: 2025-09-28
**Status**: Ready for Implementation

## 需求描述
在仓库管理页面的仓库列表表格中添加"分析时间"列，显示每个仓库最后一次代码质量分析的时间。

## 具体要求

### 显示逻辑
- **未分析**: 显示"未分析"（灰色文字）
- **分析中**: 显示"分析中..."（蓝色文字）
- **最近分析** (24小时内): 显示相对时间"2小时前"（绿色文字）
- **之前分析** (24小时外): 显示绝对日期"2025-09-27"（正常文字）

### 技术要点
- 使用现有的`Repository.last_analysis`字段（已存在）
- API已返回该字段，无需后端修改
- 在表格头部添加"分析时间"列
- JavaScript格式化日期显示

### 验收标准
1. 表格新增"分析时间"列
2. 正确显示4种分析状态
3. 日期格式符合中文习惯
4. 响应式设计正常工作
