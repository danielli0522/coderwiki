# 智能文档生成流程图

## 详细流程说明

### 1. 用户交互阶段

- **触发点**: 用户在仓库管理页面点击"生成文档"按钮
- **前端处理**: 发送 POST 请求到后端 API
- **参数传递**: 包含仓库 ID、文档类型、输出格式等配置

### 2. 后端验证阶段

- **身份验证**: 验证用户登录状态
- **权限检查**: 确认用户对指定仓库的访问权限
- **参数验证**: 检查请求参数的完整性和有效性

### 3. 任务创建阶段

- **文档记录**: 在数据库中创建文档记录
- **任务记录**: 创建异步任务记录
- **状态初始化**: 设置初始状态为 pending

### 4. BMAD 智能生成阶段

#### 4.1 Claude Code 会话管理

- **会话创建**: 通过 Claude Code SDK 创建新的工作会话
- **环境配置**: 设置工作环境和参数
- **错误处理**: 会话创建失败时的重试机制

#### 4.2 BMAD 代理团队配置

- **代理加载**: 加载预定义的 BMAD 代理配置
- **团队组建**: 配置 5 个核心代理的协作关系
- **工作流定义**: 设置代理间的执行顺序和依赖关系

#### 4.3 代码库上传

- **文件扫描**: 扫描本地代码库结构
- **文件过滤**: 过滤不需要的文件类型
- **批量上传**: 将代码文件上传到 Claude Code 会话

#### 4.4 代理协作执行

**代码分析师 Alex (20%进度)**

- 扫描代码库结构
- 识别技术栈和依赖关系
- 分析代码质量和复杂度
- 生成代码分析报告

**架构分析师 (40%进度)**

- 分析系统架构模式
- 识别组件间关系
- 评估架构质量
- 生成架构分析报告

**流程分析师 Jordan (60%进度)**

- 分析复杂业务流程
- 创建序列图和流程图
- 识别关键业务逻辑
- 生成流程分析报告

**问题解决专家 Dr. Morgan (80%进度)**

- 诊断潜在问题和风险点
- 预测可能的故障场景
- 生成解决方案建议
- 创建故障排除指南

**文档工程师 Maya (90%进度)**

- 整合所有分析结果
- 应用文档模板和格式
- 生成最终技术文档
- 进行质量验证和优化

### 5. 结果处理阶段

- **结果聚合**: 整合所有代理的输出结果
- **文档格式化**: 应用统一的文档格式
- **质量检查**: 验证文档的完整性和准确性
- **数据持久化**: 保存文档到数据库

### 6. 状态更新阶段

- **任务状态**: 更新为 completed
- **文档状态**: 更新为 published
- **进度完成**: 设置为 100%
- **时间戳**: 记录完成时间

### 7. 用户通知阶段

- **实时通知**: 通过 WebSocket 或轮询通知用户
- **结果展示**: 在前端显示生成的文档
- **下载选项**: 提供文档下载功能

## 关键技术点

### 1. 异步处理

- 使用 Python threading 进行异步任务处理
- 避免阻塞用户界面
- 支持长时间运行的文档生成任务

### 2. 进度监控

- 实时更新任务进度
- 前端轮询机制获取最新状态
- 详细的进度百分比显示

### 3. 错误处理

- 多层级的错误捕获和处理
- 自动重试机制
- 优雅的降级处理

### 4. 代理协作

- 基于 BMAD 方法的代理团队协作
- 每个代理专注于特定领域
- 结果聚合和冲突解决

### 5. 会话管理

- Claude Code 会话的生命周期管理
- 资源清理和释放
- 会话复用和优化

## 性能优化

### 1. 并发处理

- 支持多用户同时生成文档
- 独立的 Claude Code 会话
- 数据库连接池管理

### 2. 缓存策略

- 代理配置缓存
- 模板缓存
- 结果缓存

### 3. 资源管理

- 内存使用优化
- 文件上传优化
- 会话资源清理

这个流程图完整展示了从用户点击按钮到最终文档生成的整个技术实现过程，重点突出了 BMAD 代理系统的协作机制和智能文档生成的核心流程。
