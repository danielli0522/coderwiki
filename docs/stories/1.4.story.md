### Data Models
复用Story 1.3中创建的User模型：
- id: 主键
- username: 用户名
- email: 邮箱
- password_hash: 加密密码
- created_at: 创建时间
- updated_at: 更新时间
- is_active: 是否激活

[Source: docs/architecture/data-models.md]

### API Specifications
登录API端点：
- POST /api/auth/login
- POST /api/auth/logout
- GET /api/auth/status

登录请求体：
```json
{
    "username": "string",
    "password": "string"
}
```

登录成功响应：
```json
{
    "success": true,
    "user": {
        "id": 1,
        "username": "string",
        "email": "string"
    }
}
```

登录状态响应：
```json
{
    "logged_in": true,
    "user": {
        "id": 1,
        "username": "string",
        "email": "string"
    }
}
```

[Source: docs/architecture/rest-api-spec.md]
