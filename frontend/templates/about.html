关于 CoderWiki

项目简介

CoderWiki 是一个智能代码文档生成与管理平台，旨在帮助开发团队更好地管理和维护代码文档。

主要功能

  • 智能代码分析
  • 自动文档生成
  • 仓库管理
  • 文档版本控制
  • 团队协作

技术栈

  • 后端: Python Flask
  • 前端: HTML, CSS, JavaScript
  • 数据库: MySQL
  • AI集成: Claude Code

版本信息

当前版本: 1.0.0

最后更新: 2025年8月
