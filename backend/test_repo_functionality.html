仓库管理功能测试

测试步骤：
  1. 打开浏览器开发者工具（F12）
  2. 切换到Console（控制台）标签页
  3. 访问 http://localhost:5557/repositories
  4. 在控制台中运行以下测试命令
快速测试脚本

在浏览器控制台中依次运行以下命令：

1. 检查RepositoryManager是否加载
typeof repositoryManager !== 'undefined' && repositoryManager ? console.log('✅ RepositoryManager已加载') : console.log('❌ RepositoryManager未加载')
2. 查找添加仓库按钮
document.querySelectorAll('[data-bs-target="#addRepositoryModal"]').length
3. 模拟点击添加仓库按钮
document.querySelector('[data-bs-target="#addRepositoryModal"]')?.click()
4. 检查模态框是否打开
setTimeout(() => {
  const modal = document.getElementById('addRepositoryModal');
  console.log('模态框显示状态:', window.getComputedStyle(modal).display !== 'none');
}, 500)
5. 测试表单输入
const urlInput = document.getElementById('repoUrl');
if (urlInput) {
  urlInput.value = 'https://github.com/microsoft/vscode';
  urlInput.dispatchEvent(new Event('input', { bubbles: true }));
  console.log('URL输入成功:', urlInput.value);
}
6. 测试API调用
fetch('/api/repositories')
  .then(response => response.json())
  .then(data => console.log('API响应:', data.success, '仓库数量:', data.repositories?.length))
