# Claude Code SDK Demo 技术设计文档

## 项目概述

这是一个基于Claude Code SDK的演示项目，展示了如何在Web应用中集成Claude AI功能。

## 文件结构

项目包含以下关键文件：

### 1. index.html
- 大小: 2680 字符
- 功能: 主页面HTML结构
- 特征: 包含Claude相关代码

### 2. app.js
- 大小: 10268 字符
- 功能: 主要JavaScript应用逻辑
- 特征: 包含SDK集成

### 3. claude-browser-sdk.js
- 大小: 7563 字符
- 功能: Claude浏览器SDK
- 特征: 包含SDK实现

## 技术栈

- **前端**: HTML5, CSS3, JavaScript
- **AI集成**: Claude Code SDK
- **架构**: 单页面应用 (SPA)

## 功能特性

1. Claude AI集成
2. 实时对话功能
3. 响应式设计
4. 模块化代码结构

## 部署说明

1. 确保Claude API密钥配置正确
2. 启动Web服务器
3. 访问index.html页面

## 总结

这个演示项目成功展示了Claude Code SDK在Web应用中的集成方式，为开发者提供了完整的参考实现。
