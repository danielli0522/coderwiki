Archon 的外部连接

Archon 不是闭门造车。它需要和外部世界打交道——调用 AI 模型来「思考」，搜索网络来获取信息，访问数据库来存储结果。这些外部连接就像一个公司的各个供应商——公司本身不生产原料，但需要和供应商合作才能运作。

如何调用 AI 模型？

Archon 通过 API 调用 AI 模型。这是 Archon 获取「智能」的方式。

CODE 

from langchain_openai import ChatOpenAI

llm = ChatOpenAI(
    model="gpt-4o",
    temperature=0.7,
    api_key=settings.api_key
)

response = llm.invoke(prompt)

⚠️
常见陷阱：API 密钥安全

API 密钥就像你的银行卡密码——绝对不能公开分享。Archon 使用 环境变量 来安全地存储密钥，而不是直接写在代码里。

工具系统：给 Agent 装上「手」

AI Agent 只有「大脑」（LLM）是不够的。它还需要「手」——也就是能执行具体操作的工具。Archon 生成 Agent 时，会为它配备合适的工具。

🌐

网页搜索

让 Agent 能搜索互联网获取最新信息。就像给它装了一个浏览器。

📁

文件操作

让 Agent 能读写文件。就像给它装了一双手，能翻阅和编辑文档。

🔗

API 调用

让 Agent 能与外部服务通信。就像给它装了一部电话，能联系其他系统。

🗄️

数据库查询

让 Agent 能存取结构化数据。就像给它配了一个档案柜。

定义一个工具

下面是如何用代码定义一个工具——这是告诉 Agent「你有这个能力」的方式。

CODE 

from langchain_core.tools import tool

@tool
def search_web(query: str) -> str:
    """搜索互联网获取信息"""
    results = search_api.search(query)
    return results.to_string()
