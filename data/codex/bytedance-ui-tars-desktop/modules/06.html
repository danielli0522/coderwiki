Agent TARS：从桌面到终端

如果说 UI-TARS Desktop 是一个有界面的应用，那 Agent TARS 就是一把瑞士军刀——CLI、Web UI、Server 模式，随需应变。

Agent TARS 的三种使用方式

Agent TARS 不只是桌面应用——它同时提供了 CLI、Web UI 和 Server 模式。

CODE 

# 一行命令启动 Agent TARS
npx @agent-tars/cli@latest

# 指定模型和 API Key
agent-tars \
  --provider anthropic \
  --model claude-3-7-sonnet-latest \
  --apiKey your-api-key

# 或用国内模型
agent-tars \
  --provider volcengine \
  --model doubao-1-5-thinking-vision-pro \
  --apiKey your-api-key
PLAIN ENGLISH

用 npx 一行命令启动，不需要安装任何东西...

这会打开一个 Web UI 界面...

指定使用 Anthropic 的 Claude 模型...

使用 claude-3-7-sonnet 这个视觉语言模型...

传入你的 API 密钥进行身份验证...

也可以用国内的火山引擎模型...

使用豆包视觉思考模型——国内用户更友好...

同一个框架，多个模型提供商，按需切换。

浏览器控制策略

Agent TARS 控制浏览器有三种模式——这是它区别于其他 GUI Agent 的核心能力。

👁️

GUI Agent 模式

像人一样"看"浏览器截图，理解视觉元素并操作。通用性最强，但速度较慢（每步需要截图+模型推理）

🌲

DOM 模式

直接读取网页的 DOM 结构，精确定位元素。速度快，准确性高，但只能用于浏览器

🔄

混合模式（Hybrid）

智能切换：简单操作用 DOM 快速执行，复杂界面用视觉理解。兼顾速度和通用性
