Like a Package Delivery System

When you place an order on your phone, what happens behind the scenes? The order goes to the warehouse → the warehouse picks the item → the courier picks it up → it is delivered to your door. Each step has a clear role and data flow direction.

When you type /comps Apple Inc., a similar chain of events kicks off:

1
User inputs a command

You type /comps Apple Inc. in Claude

2
Command file is activated

comps.md tells Claude what information to ask the user (target company, peer list, time range)

3
Skill file provides methodology

comps-analysis/SKILL.md guides Claude on how to perform the analysis: query MCP data sources first, then build tables, then calculate multiples

4
Fetch data via MCP connectors

Claude accesses data sources like S&P and FactSet through channels configured in .mcp.json

5
Deliver results to the user

Generate an Excel spreadsheet or PowerPoint deck with the complete comparable company analysis

MCP: The 'Universal Data Interface' for AI

MCP is an open protocol designed by Anthropic that enables AI to securely access external data. Here are the data providers connected in this project:

.mcp.json 

{
  "mcpServers": {
    "daloopa": { "type": "http",
      "url": "https://mcp.daloopa.com/server/mcp" },
    "morningstar": { "type": "http",
      "url": "https://mcp.morningstar.com/mcp" },
    "sp-global": { "type": "http",
      "url": "https://kfinance.kensho.com/.../mcp" }
  }
}
Plain English

This is an MCP configuration file registering three data channels.

Daloopa — a tool that automatically extracts financial data from earnings report PDFs.

Morningstar — an authoritative source for fund and investment data.

S&P Global (Kensho) — market data, valuation multiples, and company financials.

Each connector is an HTTP endpoint the AI uses to retrieve data.

Listen In on Their Discussion

The skill files contain strict data priority rules. When Claude needs data, the Skill file tells it which source to check first:

🤖
Claude (Agent)

The user wants a comparable company analysis for Apple. I need financial data. Should I search Google first?

📘
comps-analysis Skill

Stop! Do not use web search. Prioritize MCP data sources: check S&P Kensho first, then FactSet.

🤖
Claude (Agent)

Why can't I use search? Can't search results also find the data?

📘
comps-analysis Skill

Search results may be outdated, inaccurate, and lack audit trails. MCP sources provide institutional-grade data with proper citation. You cannot make investment decisions based on randomly googled numbers.

🤖
Claude (Agent)

Got it. What if all MCP sources are unavailable?

📘
comps-analysis Skill

Only then can you use Bloomberg Terminal or public SEC EDGAR filings. Web search must never be used as a primary data source.

💬
💡
Core Insight

Skill files are not just 'how-to manuals' — they also contain data quality guardrails. By embedding data priority rules in the Skill, even if the AI instinctively tries to take a shortcut (searching the web), the skill rules pull it back. This approach of 'encoding best practices into rules' is the key to making AI reliable in professional domains.
