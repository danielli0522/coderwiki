What Is This?

A plugin system that lets AI handle tasks for financial professionals

Imagine This Scenario

You are an analyst at an investment bank. At 9 AM you receive a task: perform a comparable company analysis on a company, due by 2 PM.

Traditional process: Open the Bloomberg terminal → manually copy data → build an Excel spreadsheet → calculate multiples → make a PPT → double-check everything. At least 4 hours.

With this project, you just type /comps target company name, and the AI handles the entire process automatically.

Anthropic Financial Services Plugins

This is a collection of plugins officially open-sourced by Anthropic, designed specifically for the financial industry. It enables Claude to work like a financial analyst — building valuation models, writing research reports, conducting due diligence, reconciling accounts, and more.

A

⇔
B

💡
Key Design Philosophy

The same system prompt, the same set of skill files — whether for personal use or enterprise deployment, the AI behaves identically. This 'one source, two packages' pattern is known in software engineering as 'Single Source of Truth.'

Check Out the Directory Structure

The file organization of the entire project looks like this:

Directory Structure 

plugins/
  agent-plugins/    # 10 named Agents
  vertical-plugins/ # 7 industry skill packs
  partner-built/    # Partner plugins (LSEG, S&P)
managed-agent-cookbooks/  # API deployment templates
scripts/          # Deployment & validation scripts
Plain English

plugins/ — All installable plugins live here

agent-plugins/ — Complete 'AI employees,' each responsible for one business line

vertical-plugins/ — 'Skill manuals' broken down by industry

partner-built/ — Integration plugins written by data providers themselves

cookbooks/ — Configuration files for deploying Agents to enterprise backends

scripts/ — Automated deployment and code quality check tools
