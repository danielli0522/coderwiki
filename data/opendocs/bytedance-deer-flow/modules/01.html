DeerFlow 是什么

一个字节跳动开源的超级 Agent 编排框架——把子智能体、记忆、沙箱和技能组合在一起，帮你完成几乎任何任务

想象你在用 DeerFlow

你在浏览器里打开 DeerFlow 的聊天界面，输入一句话：

💬
用户输入

"帮我研究一下 2026 年 AI Agent 框架的市场格局，生成一份带图表的报告"

然后你坐下来喝杯咖啡。几分钟后，你收到了一份完整的分析报告——包含数据搜索、竞品对比、趋势图表。这不是一个 LLM 在单打独斗，而是一整支"AI 团队"在并行工作。

它到底做了什么？

DeerFlow 2.0（Deep Exploration and Efficient Research Flow）是字节跳动开源的 超级 Agent 编排框架。它不是一个聊天机器人，而是一个"AI 团队的项目经理"。

1
接收你的指令

通过网页聊天、飞书、钉钉、Telegram 等渠道输入任务

2
拆解成子任务

Lead Agent 把复杂任务分解成多个可以并行执行的子任务

3
派发子 Agent 执行

每个子 Agent 在独立的沙箱环境里搜索、编码、分析

4
汇总结果返回给你

Lead Agent 把所有子 Agent 的结果整合成一份连贯的回答

💡
关键洞察

DeerFlow 2.0 是一次彻底重写，和 v1 没有共用代码。v1 是一个 Deep Research（深度搜索）框架，v2 进化成了一个通用的"超级 Agent 驾驶舱"——可以接入各种工具、记忆、技能，完成几乎任何任务。

源码入口：一切从这里开始

整个后端的入口在 backend/app/gateway/app.py，这段代码创建了 FastAPI 应用并注册了所有 API 路由：

CODE 

def create_app() -> FastAPI:
    app = FastAPI(
        title="DeerFlow API Gateway",
        lifespan=lifespan,
    )
    app.add_middleware(AuthMiddleware)
    app.include_router(models.router)
    app.include_router(memory.router)
    app.include_router(skills.router)
    app.include_router(agents.router)

创建 FastAPI 应用实例，这是整个后端的"总入口"。

设置应用标题，用于 API 文档展示。

注册模型管理路由——查询可用的大模型列表。

注册技能管理路由——查询和配置 Agent 技能。

注册 Agent 管理路由——创建和配置自定义 Agent。
